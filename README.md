# DevOps Assignments

This repository contains a collection of DevOps assignments covering various technologies and practices including Docker, Kubernetes, CI/CD, microservices, and cloud deployments.

## Project Structure

### ASSIGN 1 - Python Flask Application with Docker & Jenkins
- **Technology**: Python, Flask, Docker, Jenkins
- **Components**: 
  - Flask application (`app.py`)
  - Dockerfile for containerization
  - Jenkinsfile for CI/CD pipeline
  - Requirements file for Python dependencies

### ASSIGN 2 - Angular Frontend Application
- **Technology**: Angular, TypeScript, Docker, Docker Compose
- **Components**:
  - Angular application configuration
  - Dockerfile and docker-compose.yml for containerization
  - TypeScript configuration files

### ASSIGN 3 - Java Spring Boot Application
- **Technology**: Java, Spring Boot, Maven
- **Components**:
  - Spring Boot application (`AnushreeApplication.java`)
  - Maven configuration (`pom.xml`)
  - Jenkins pipeline configuration
  - Application properties

### ASSIGN 4 - Java Application
- **Technology**: Java, Maven
- **Components**:
  - Compiled Java classes
  - Maven project configuration
  - Test files

### ASSIGN 5 - Spring Boot Demo Application
- **Technology**: Java, Spring Boot, Docker
- **Components**:
  - Spring Boot demo application
  - HelloController REST endpoint
  - Dockerfile for containerization
  - Maven wrapper

### ASSIGN 6 - Kubernetes Deployment
- **Technology**: Kubernetes, Docker, Node.js, Kind
- **Components**:
  - Deployment scripts (`deploy.sh`)
  - Kubernetes configuration (`kind-config.yaml`)
  - Metrics server fix script
  - Dockerfile
  - Node.js application

### ASSIGN 7 - MongoDB & Mongo Express on Kubernetes
- **Technology**: Kubernetes, MongoDB, Mongo Express
- **Components**:
  - MongoDB deployment and service
  - Mongo Express deployment and service
  - Persistent Volume Claims
  - ConfigMaps and Secrets
  - Namespace configuration

### ASSIGN 8 - Microservices Architecture
- **Technology**: Node.js, Docker, Kubernetes, API Gateway
- **Components**:
  - Frontend application (HTML/nginx)
  - API Gateway
  - Posts microservice
  - Users microservice
  - Kubernetes deployments and services
  - ConfigMaps and Secrets

### ASSIGN 9 - Apache Web Server on Kubernetes
- **Technology**: Kubernetes, Apache HTTP Server
- **Components**:
  - Apache deployment configuration
  - Service configuration
  - Documentation images

## Prerequisites

- Docker and Docker Compose
- Kubernetes cluster (minikube, kind, or cloud provider)
- Java JDK (for Java projects)
- Node.js and npm (for Node.js projects)
- Python 3.x (for Python projects)
- Maven (for Java projects)
- Jenkins (for CI/CD assignments)

## Getting Started

Each assignment directory contains its own README or documentation with specific instructions. Navigate to the respective assignment folder for detailed setup and deployment instructions.

## Technologies Used

- **Containerization**: Docker, Docker Compose
- **Orchestration**: Kubernetes
- **CI/CD**: Jenkins
- **Languages**: Python, Java, TypeScript/JavaScript
- **Frameworks**: Flask, Spring Boot, Angular
- **Databases**: MongoDB
- **Web Servers**: Nginx, Apache
